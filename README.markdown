Random installation tips:
-------------------------
1. `git clone git://github.com/nazgob/dotfiles.git`  
2. `cd dotfiles`  
3. `git submodule init`  
4. `git submodule update`   
5. `ln -s ~/Projects/dotfiles/.gemrc ~/.gemrc`
