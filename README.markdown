Random installation tips:
============

- git clone git://github.com/nazgob/dotfiles.git
- cd dotfiles
- git submodule init
- git submodule update

- cd .vim/bundle/command-t/ruby/command-t/
- ruby extconf.rb
- make

- ln -s ~/Projects/dotfiles/.vimrc ~/.vimrc
- ln -s ~/Projects/dotfiles/.vim ~/.vim
